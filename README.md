# レールタイクーン (RailTycoon)

まっさらな土地に駅と線路を配置し、列車を走らせて乗客と街を育てる鉄道経営シミュレーション。
Unity 6 (6000.3.20f1) 製プロトタイプ。シーン・UI・メッシュは全てコード生成。

## 遊び方
1. **駅** モード: 対応両数(2〜10両)・ホーム面数(1〜4)・番線数を決めて地面をタップ → 「ここに建設」
2. **線路** モード: 駅を2つタップして複線で接続
3. **列車** モード: 編成を選び、駅を順にタップして経路を作成 → 「発車!」
4. 乗客が増えると駅周辺に街が発展し、運賃収入で路線を拡大

## 車種
- 京王5000系(10両)
- 名鉄2000系ミュースカイ(4/8両)
- 名鉄2200系(6両)
- 名鉄6000系(2/4両)

停車には列車の両数以上の「対応両数」を持つ駅が必要。番線数は同時に停車・進入できる列車数の上限。

## 開発メモ(バッチ実行)
- `SceneSetup.CreateAll`: マテリアル・Main.unity生成
- `Snapshot.Run`: デモ配置のスクリーンショットを `Snapshots/` に出力
- `WebGLBuild.Run`: `Builds/WebGL` へビルド(Brotli + decompressionFallback)
